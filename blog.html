sessionStorage is similar to localStorage, however, the difference is that while data in localStorage doesn't expire, data in sessionStorage is cleared when the page session ends.

A global variable has global scope. A global variable is accessible from anywhere in the code. On the other hand the local scope contains things defined inside code blocks

The Event Loop has one simple job to monitor the Call Stack and the Callback Queue. If the Call Stack is empty, the Event Loop will take the first event from the queue and will push it to the Call Stack, which effectively runs it. Such an iteration is called a tick in the Event Loop.
